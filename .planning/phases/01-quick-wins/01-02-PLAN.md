<task type="auto">
  <name>Task 1: Update IntegrationsStatus badge to use consistent styling</name>
  <files>frontend/components/dashboard/IntegrationsStatus.vue</files>
  <action>
    Le composant utilise PrimeVue Badge avec severity="secondary" pour "Déconnecté".

    Ajouter une classe custom au Badge pour améliorer le contraste:

    Trouver:
    ```vue
    <Badge
      :value="integration.is_connected ? 'Connecté' : 'Déconnecté'"
      :severity="integration.is_connected ? 'success' : 'secondary'"
    />
    ```

    Remplacer par (ajouter class pour le style disconnected):
    ```vue
    <Badge
      :value="integration.is_connected ? 'Connecté' : 'Déconnecté'"
      :severity="integration.is_connected ? 'success' : 'secondary'"
      :class="!integration.is_connected ? 'badge-status-disconnected' : ''"
    />
    ```

    Note: On garde severity pour la compatibilité PrimeVue mais on override avec notre classe.
  </action>
  <verify>grep -q "badge-status-disconnected" frontend/components/dashboard/IntegrationsStatus.vue</verify>
  <done>Badge IntegrationsStatus utilise la classe badge-status-disconnected</done>
</task>
