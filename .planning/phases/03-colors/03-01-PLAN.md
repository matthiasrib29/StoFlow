<task type="auto">
  <name>Task 1: Add link color utilities to design system</name>
  <files>frontend/assets/css/design-system.css</files>
  <action>
    Ajouter une section pour les liens dans design-system.css après la section "11. PRIMEVUE OVERRIDES":

    ```css
    /* ============================================
       12. LIENS
       ============================================ */

    /* Lien standard - Utiliser pour tous les liens texte */
    .link {
      @apply text-primary-600 hover:text-primary-700 transition-colors;
    }

    .link-underline {
      @apply text-primary-600 hover:text-primary-700 underline transition-colors;
    }

    /* Lien secondaire (gris) */
    .link-muted {
      @apply text-gray-500 hover:text-secondary-900 transition-colors;
    }
    ```

    Cela standardise les couleurs de liens à travers l'application.
  </action>
  <verify>grep -q "\.link {" frontend/assets/css/design-system.css</verify>
  <done>Classes .link, .link-underline, .link-muted définies</done>
</task>
