<task type="auto">
  <name>Task 1: Add spacing utility classes to design system</name>
  <files>frontend/assets/css/design-system.css</files>
  <action>
    Ajouter une section "16. SPACING STANDARDS" après la section "15. ICON SIZE STANDARDS":

    ```css
    /* ============================================
       16. SPACING STANDARDS
       ============================================ */

    /* Standard spacing values (4px base unit):
       - xs: 4px (gap-1)
       - sm: 8px (gap-2)
       - md: 16px (gap-4)
       - lg: 24px (gap-6)
       - xl: 32px (gap-8)
    */

    /* Form field spacing - Standardized to 8px (gap-2) */
    .form-field {
      @apply space-y-2;
    }

    /* Grid gaps for cards */
    .grid-cards {
      @apply grid gap-6;
    }

    /* Sidebar submenu indentation */
    .sidebar-submenu {
      @apply ml-4 pl-4 border-l-2 border-gray-100;
    }

    /* Table cell padding */
    .table-cell-standard {
      @apply px-4 py-3;
    }
    ```

    Cela documente et standardise les espacements pour référence future.
  </action>
  <verify>grep -q "SPACING STANDARDS" frontend/assets/css/design-system.css</verify>
  <done>Classes utilitaires d'espacement définies dans le design system</done>
</task>
