  <name>Task 1: Add global heading typography enforcement</name>
  <files>frontend/assets/css/design-system.css</files>
  <action>
    Le design-system.css a déjà une règle pour h1-h6 mais elle n'est pas assez spécifique.

    Ajouter/renforcer les règles typographiques dans la section "7. TYPOGRAPHIE":

    1. S'assurer que les headings utilisent bien font-display:
    ```css
    /* Headings génériques - Enforce font-display */
    h1 { @apply font-display text-3xl font-bold tracking-tight; }
    h2 { @apply font-display text-2xl font-bold tracking-tight; }
    h3 { @apply font-display text-xl font-semibold tracking-tight; }
    h4 { @apply font-display text-lg font-semibold; }
    h5 { @apply font-display text-base font-semibold; }
    h6 { @apply font-display text-sm font-semibold; }
    ```

    Remplacer la règle existante `h1, h2, h3, h4, h5, h6 { @apply font-display tracking-tight; }` par ces règles plus spécifiques.

    Note: Ne pas ajouter de couleur car les composants définissent leurs propres couleurs.
  </action>
  <verify>grep -A2 "^h1 {" frontend/assets/css/design-system.css</verify>
  <done>Règles h1-h6 avec font-display et font-weight définies</done>

<task type="auto">
  <name>Task 2: Add PrimeVue DataTable header styling</name>
  <files>frontend/assets/css/design-system.css</files>
  <action>
    Ajouter une section pour les overrides PrimeVue DataTable dans design-system.css.

    Ajouter après la section "10. ETATS VIDES":

    ```css
    /* ============================================
       11. PRIMEVUE OVERRIDES
       ============================================ */

    /* DataTable header styling */
    .p-datatable .p-datatable-thead > tr > th {
      @apply font-display font-semibold text-secondary-700 bg-gray-50;
    }

    /* DataTable body text */
    .p-datatable .p-datatable-tbody > tr > td {
      @apply font-body text-secondary-900;
    }
    ```

    Cela garantit que tous les DataTable ont des headers cohérents avec le design system.
  </action>
  <verify>grep -q "p-datatable-thead" frontend/assets/css/design-system.css</verify>
