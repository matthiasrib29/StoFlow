<task type="auto">
  <name>Task 1: Add PrimeVue form input height standardization</name>
  <files>frontend/assets/css/design-system.css</files>
  <action>
    Ajouter une section pour les overrides de formulaires PrimeVue après la section "12. LIENS":

    ```css
    /* ============================================
       13. PRIMEVUE FORM OVERRIDES
       ============================================ */

    /* Input height standardization - 44px (11 * 4px) */
    .p-inputtext,
    .p-select,
    .p-dropdown,
    .p-multiselect,
    .p-textarea {
      @apply h-11;
    }

    /* InputText specific styling */
    .p-inputtext {
      @apply rounded-lg border-gray-300 focus:border-primary-400 focus:ring-2 focus:ring-primary-100;
    }

    /* Select/Dropdown specific styling */
    .p-select,
    .p-dropdown {
      @apply rounded-lg border-gray-300;
    }

    .p-select:focus,
    .p-dropdown:focus,
    .p-select.p-focus,
    .p-dropdown.p-focus {
      @apply border-primary-400 ring-2 ring-primary-100;
    }
    ```

    Cela garantit que tous les inputs PrimeVue ont une hauteur uniforme de 44px.
  </action>
  <verify>grep -q "p-inputtext" frontend/assets/css/design-system.css && grep -q "h-11" frontend/assets/css/design-system.css</verify>
  <done>PrimeVue form inputs standardisés à 44px de hauteur</done>
</task>

<task type="auto">
  <name>Task 2: Add border-radius utility classes</name>
  <files>frontend/assets/css/design-system.css</files>
  <action>
    Ajouter des classes utilitaires pour le border-radius standardisé après la section "13. PRIMEVUE FORM OVERRIDES":

    ```css
    /* ============================================
       14. BORDER RADIUS STANDARDS
       ============================================ */

    /* Standard radius values from design system:
       - Cards/containers: rounded-xl (16px)
       - Buttons/inputs: rounded-lg (12px)
       - Pills/badges: rounded-full
       - Small elements: rounded-md (8px)
    */

    /* Utility class for card radius */
    .radius-card {
      @apply rounded-xl;
    }

    /* Utility class for button/input radius */
    .radius-button {
      @apply rounded-lg;
    }

    /* Override PrimeVue Button radius to match our standard */
    .p-button {
      @apply rounded-lg;
    }
    ```

    Ces classes documentent et appliquent le standard de border-radius.
  </action>
  <verify>grep -q "radius-card" frontend/assets/css/design-system.css</verify>
  <done>Classes utilitaires radius-* définies, PrimeVue Button standardisé</done>
</task>
